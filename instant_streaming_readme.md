### **Mode 2: Peer-to-Peer Streaming**
**What it does:**
- Generates a shareable link
- Viewers watch your stream live in their browser
- Direct connection - no relay server

**How to use:**
1. Select "Peer-to-Peer" from protocol dropdown
2. Start capturing ‚Üí Start streaming
3. Copy the shareable link that appears
4. Send link to viewers!

**Best for:**
- Small audiences (1-10 viewers)
- Same network streaming (e.g., LAN parties)
- Private streams to friends/family
- Testing before going live on platforms

**Limitations:**
- Works best on same network
- Limited by your upload bandwidth
- May need port forwarding for remote viewers

## üîß **ADVANCED OPTIONS:**

### **Want to stream to Twitch/YouTube?**

You'll need a relay server. Three options:

**Option 1: Use Cloudflare Stream** (easiest)
- Sign up at dash.cloudflare.com
- Get WHIP endpoint + API token
- Select "WebRTC-WHIP" protocol
- Enter your credentials

**Option 2: Run WebSocket bridge** (advanced)
- Requires Node.js + FFmpeg
- See server setup guide in the app

**Option 3: Use OBS** (recommended for platforms)
- Use OBS to stream to Twitch/YouTube
- Use this app for recording only
- Best of both worlds!
