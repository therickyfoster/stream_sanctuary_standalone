1. Uses Screen Capture API to grab your screen
2. Uses MediaRecorder API to encode video
3. Chunks saved in browser memory
4. Creates Blob when stopped
5. Auto-downloads via download link

Peer-to-Peer Mode:
1. Creates WebRTC peer connection
2. Uses public STUN servers (Google)
3. Generates SDP offer with ICE candidates
4. Encodes offer as shareable URL
5. Viewers connect directly to you

Q: Do I need to install anything?
A: No! Just open the HTML file in a browser.

Q: Do I need a server?
A: No! Both default modes work without any server.

Q: Can I stream to Twitch/YouTube?
A: Use Browser Recording mode, then upload the video file.
   For live streaming to platforms, you'd need a relay server.

Q: Does it work offline?
A: Yes! Browser Recording mode works 100% offline.

Q: Where's my video saved?
A: Auto-downloads to your browser's download folder.

Q: What video format?
A: WebM (VP9 + Opus). Widely supported, can convert to MP4.

Q: File size limit?
A: Only limited by your available disk space.

Q: Can multiple people watch live?
A: Yes, with P2P mode! Best for 1-10 viewers on same network.

Q: Is this really zero setup?
A: YES! Absolutely zero setup. No tricks. Just open and use.
